# 🍦 Ice Cream Sales Prediction

A Machine Learning mini project built from scratch using **Linear Regression** and **Gradient Descent** in Python.

## Features

- Reads dataset from CSV
- Implements Linear Regression without Scikit-Learn
- Computes Cost Function
- Computes Gradient
- Uses Gradient Descent to learn parameters
- Predicts Ice Cream Sales based on Temperature

## Technologies

- Python
- NumPy
- Pandas
- Matplotlib

## Project Structure

```
IceCream-Sales-Prediction/
│
├── icecream_sales.csv
├── main.py
├── model.py
├── README.md
└── requirements.txt
```

## Run

```bash
pip install -r requirements.txt
python main.py
```

```
Enter Temperature (°C): 35
```
